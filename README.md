# demo-eleventy-from-scratch-blog

Companion code to the “6 Minutes to Build a Blog from Scratch” tutorial: https://www.youtube.com/watch?v=kzf9A9tkkl4

Run this:

```
npm install
npm start
```

Navigate to `http://localhost:8080/` in your web browser.
